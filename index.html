Utility-first CSS,

KadakCSS scans your DOM, reads kc-* class names, and applies styles dynamically — no PostCSS, no bundler, no magic. Just JavaScript the way it was meant to be.

JavaScript API

// Add a custom color at runtime
KadakCSS.addColor('mango', '#ffb347');

// Add a custom utility class
KadakCSS.addUtility('kc-card', {
  padding: '24px',
  borderRadius: '12px',
});

// Re-scan after programmatic DOM changes
KadakCSS.refresh();

// Enable debug warnings
KadakCSS.debug = true;
